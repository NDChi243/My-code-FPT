## 📝 Hướng dẫn làm bài

1. **Đọc đề** trong file `README.md` của từng slot
2. **Tự code** — không nhìn solution trước
3. **Kiểm tra** — compile và chạy thử với các test case mẫu
4. **So sánh** — mở file trong `solution/` để xem đáp án tham khảo
5. **Rút kinh nghiệm** — ghi chú lại những lỗi sai hoặc cách làm hay hơn

> 💡 **Mẹo:** Mỗi bài đều có `Input validation` — hãy tập thói quen kiểm tra dữ liệu đầu vào ngay từ bây giờ!
