# Slot 02 — Bài tập: Hệ thống máy tính & Quy trình phát triển

## Bài 1: Giải thích chương trình (explain_code.c)
Cho đoạn code sau:
```c
#include <stdio.h>

int main() {
    printf("Hello, PRF192!\n");
    return 0;
}
```
Viết chương trình **giống hệt** và thêm comment (bằng tiếng Việt) giải thích ý nghĩa từng dòng.

---

## Bài 2: Liệt kê linh kiện (computer_parts.c)
Viết chương trình in ra bảng các thành phần máy tính và chức năng của chúng:

```
=== CAC THANH PHAN MAY TINH ===
1. CPU    - Bo xu ly trung tam
2. RAM    - Bo nho tam thoi
3. HDD/SSD- O dia cung
4. Input  - Thiet bi nhap (ban phim, chuot)
5. Output - Thiet bi xuat (man hinh, may in)
```

---

## Bài 3: Quy trình phát triển (dev_process.c)
Viết chương trình in ra 5 bước của quy trình phát triển phần mềm dạng số thứ tự:
```
1. Phan tich
2. Thiet ke
3. Code
4. Test
5. Debug
```

**Yêu cầu:** Mỗi bước in trên một dòng riêng. Dùng `printf` với `\n`.
