# Slot 11-12 — Bài tập: Ôn tập CLO1, CLO2, CLO3

## Bài 1: Tổng giai thừa (factorial_sum.c)
Nhập `n` (nguyên dương). Tính `S = 1! + 2! + 3! + ... + n!`.

**Ví dụ:** `n = 4` → `S = 1 + 2 + 6 + 24 = 33`

**Yêu cầu:** Kiểm tra `n > 0`. Không dùng thư viện math.h.

---

## Bài 2: Ước chung lớn nhất (gcd_lcm.c)
Nhập 2 số nguyên dương `a` và `b`. Tính:
- Ước chung lớn nhất (GCD) — dùng thuật toán Euclid
- Bội chung nhỏ nhất (LCM) — `LCM = (a * b) / GCD`

**Ví dụ:** `a = 12, b = 18` → `GCD = 6, LCM = 36`

---

## Bài 3: Vẽ hình tam giác (triangle_star.c)
Nhập chiều cao `h` (nguyên dương). In hình tam giác vuông cân bằng dấu `*`:

**Input:** `Nhap h: 5`
**Output:**
```
*
**
***
****
*****
```

**Yêu cầu:** Kiểm tra `h > 0`.
