# Slot 23-24 — Bài tập: Ôn tập CLO4

## Bài 1: Menu chương trình tính toán (menu_calc.c)
Viết chương trình menu với các chức năng dùng hàm:
```
===== MENU TINH TOAN =====
1. Tinh tong cac so tu 1 den n
2. Tinh tong cac so le tu 1 den n
3. Kiem tra so nguyen to
4. Thoat
```

**Yêu cầu:** Mỗi chức năng là một hàm riêng. Dùng `do-while` cho menu.

---

## Bài 2: Hàm kiểm tra số đối xứng (palindrome_num.c)
Viết hàm `int laSoDoiXung(int n)` kiểm tra số đọc xuôi ngược giống nhau (VD: 121, 12321).

**Input:** `Nhap n: 12321`  
**Output:** `12321 la so doi xung`

**Yêu cầu:** Không chuyển thành chuỗi, dùng vòng lặp đảo ngược số.

## Bài 3: Bội số chung nhỏ nhất (lcm_func.c)
Viết hàm `int timBCNN(int a, int b)` tìm bội chung nhỏ nhất.
Viết hàm `int timUCLN(int a, int b)` tìm ước chung lớn nhất (Euclid).

Nhập `a, b`. In ra: `UCLN(a, b) = X, BCNN(a, b) = Y`
