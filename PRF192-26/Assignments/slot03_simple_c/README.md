# Slot 03 — Bài tập: Cấu trúc chương trình C đơn giản

## Bài 1: Hello World nhiều ngôn ngữ (hello_multi.c)
Viết chương trình in "Hello World" bằng 3 ngôn ngữ khác nhau:
```
Hello World!   (English)
Xin chao!      (Vietnamese)
Bonjour!       (French)
```

---

## Bài 2: In thông tin lớp (class_info.c)
Viết chương trình khai báo biến và in ra:
- Tên môn học (chuỗi)
- Số tín chỉ (int)
- Tên giảng viên (chuỗi)
- Phòng học (chuỗi)

**Ví dụ output:**
```
Mon: PRF192 - Co so Lap trinh C
So tin chi: 3
Giang vien: Nguyen Van A
Phong: Lab-401
```

---

## Bài 3: Tính toán đơn giản (simple_math.c)
Viết chương trình khai báo hai số nguyên `a = 10` và `b = 3`, sau đó tính và in ra:
- Tổng
- Hiệu
- Tích
- Thương (dạng số thực)

**Output mong đợi:**
```
a = 10, b = 3
Tong: 13
Hieu: 7
Tich: 30
Thuong: 3.33
```
